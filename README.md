# _dLabs Docs

https://docs.underscoredlabs.com

## Installing

```bash
# https://www.mkdocs.org/
$ pip install mkdocs
$ git clone git@github.com:underscoredLabs/website-dLabs-docs.git
$ cd website-dLabs-docs
```

## Development

```bash
$ mkdocs serve
```

## Publishing
```bash
$ mkdocs gh-deploy
```
