# uToken-website

## Installing

```bash
# https://www.mkdocs.org/
$ pip install mkdocs
$ git clone git@github.com:underscoredLabs/website-uToken.git
$ cd website-uToken
```

## Development

```bash
$ mkdocs serve
```

## Publishing
```bash
$ mkdocs build
# copy contents from /site
$ git checkout website
# paste contents in this branch
```
